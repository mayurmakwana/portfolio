# Mayur Makwana - Full Stack Developer Portfolio

## 🚀 Modern & Responsive Portfolio Website

A premium, production-ready portfolio website built with cutting-edge technologies.

### 🛠️ Tech Stack

- **Framework**: Next.js 14 with React 18
- **Styling**: Tailwind CSS 3
- **Animations**: Framer Motion 10
- **Language**: TypeScript
- **Form Handling**: React Hook Form
- **Icons**: React Icons
- **Email**: Nodemailer

### 📋 Features

✅ Responsive dark theme design
✅ Glassmorphism UI components
✅ Smooth animations with Framer Motion
✅ Mobile-first approach
✅ SEO optimized
✅ Fast performance
✅ Contact form with email integration
✅ Project showcase with live demo links
✅ Skills timeline and experience section
✅ Testimonials carousel

### 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   npm start
   ```

### 📧 Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_EMAIL_JS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAIL_JS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAIL_JS_PUBLIC_KEY=your_public_key
```
